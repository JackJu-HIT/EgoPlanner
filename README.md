# FastPlanner

将EgoPlanner后端（纯净版）基于B样条的轨迹优化封装单独可用的C++文件，并将三维轨迹将为2维轨迹，适用于地面移动机器人实时局部路径规划。

# how to use ?

mkdir build 

cd build

cmake ..

make

./EgoPlanCore
