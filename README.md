# FastPlanner

将FastPlanner后端基于贝塞尔的轨迹优化封装单独可用的C++文件（纯净版本），并将三维轨迹将为2维轨迹，适用于地面小车实时规划。

# how to use ?

mkdir build 

cd build

cmake ..

make

./EgoPlanCore
